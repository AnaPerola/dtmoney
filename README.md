## 🔧 Iniciando o projeto

1 - Clone o projeto
```console
$ git clone git@github.com:AnaPerola/dtmoney.git
```
2 - Execute o comando para instalar todas as dependências
```console
$ yarn install
```
3 - Suba o projeto 
```console
$ yarn start
```

Se tudo ocorreu bem, visite: http://localhost:3000 😆
